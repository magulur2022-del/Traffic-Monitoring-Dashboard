# рҹ“Ң Project Overview

The Smart Traffic Monitoring Dashboard helps traffic administrators monitor, analyze, and manage traffic efficiently through a centralized dashboard.

The application provides real-time traffic visualization, AI-based vehicle detection, live map monitoring, smart traffic signal control, traffic analytics, reports, and system settings.

# рҹҡҖ Installation

## Clone Repository

```bash
git clone https://github.com/magulur2022-del/Traffic-Monitoring-Dashboard.git
```

## Install Frontend

```bash
cd client
npm install
npm run dev
```

## Install Backend

```bash
cd server
npm install
npm start
```

## Run AI Module

```bash
cd python-ai
pip install -r requirements.txt
python detect.py
```
